DesktopHatter Distribution Folder (macOS)

How to run:
1. Open `DesktopHatter_macos.app`.
2. If macOS shows a security warning, use one of these methods:
   - Right-click `DesktopHatter_macos.app` -> `Open` -> `Open`
   - System Settings -> Privacy & Security -> `Open Anyway`

Basic controls:
- Drag: move mascot
- Right-click: open menu
- Double-click mascot: shake + random help tip
- Menu: Dress-up / Reminders / News / Language / Quit

Save data location (default):
- `~/Library/Application Support/DesktopHatter/`

Main saved files:
- hats.json
- reminders.json
- face_style.cfg
- window_state.cfg

Uninstall:
1. Delete `DesktopHatter_macos.app`.
2. To remove all user data, also delete:
   `~/Library/Application Support/DesktopHatter/`

DesktopHatter 配布フォルダ (macOS)

起動方法:
1. `DesktopHatter_macos.app` を開きます。
2. macOS のセキュリティ警告が表示された場合:
   - `DesktopHatter_macos.app` を右クリック -> `開く` -> `開く`
   - システム設定 -> プライバシーとセキュリティ -> `このまま開く`

基本操作:
- ドラッグ: マスコットを移動
- 右クリック: メニューを開く
- ダブルクリック: ブルブルしてヒントを表示
- メニュー: きがえる / リマインダー / ニュース / 言語 / 終了

セーブデータ保存先 (デフォルト):
- `~/Library/Application Support/DesktopHatter/`

主な保存ファイル:
- hats.json
- reminders.json
- face_style.cfg
- window_state.cfg

アンインストール:
1. `DesktopHatter_macos.app` を削除します。
2. ユーザーデータも削除する場合は次も削除します:
   `~/Library/Application Support/DesktopHatter/`
