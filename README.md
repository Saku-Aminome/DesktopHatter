Thank you for using DesktopHatter.

This package contains two platform folders:
- DesktopHatter_windows/
- DesktopHatter_macos/

Please open the folder for your OS and run the app from there.

Windows:
- Run: DesktopHatter_windows.exe
- If SmartScreen appears:
  Click "More info" -> "Run anyway"

macOS:
- Run: DesktopHatter_macos.app
- If macOS blocks launch:
  Right-click the app -> Open -> Open
  or use System Settings -> Privacy & Security -> Open Anyway

Basic controls:
- Drag mascot: move position
- Right-click mascot: open menu
- Double-click mascot: shake + random help tip

Save data location:
- Windows: %APPDATA%\DesktopHatter\
- macOS: ~/Library/Application Support/DesktopHatter/

Uninstall:
- Delete the app folder for your OS
- Optional: delete save data folder above to remove all user data

Notes:
- This app may show security warnings on Windows/macOS when distributed without code signing.
- Keep the app file and .pck file together in the same folder.

====================  
  
DesktopHatter をご利用いただきありがとうございます。

この配布パッケージには、OS別のフォルダが含まれています:
- DesktopHatter_windows/
- DesktopHatter_macos/

お使いのOSのフォルダを開いて、アプリを起動してください。

Windows:
- 起動ファイル: DesktopHatter_windows.exe
- SmartScreen が表示された場合:
  「詳細情報」->「実行」を選択してください

macOS:
- 起動ファイル: DesktopHatter_macos.app
- 起動がブロックされる場合:
  アプリを右クリック -> 「開く」->「開く」
  または「システム設定」->「プライバシーとセキュリティ」->「このまま開く」

基本操作:
- マスコットをドラッグ: 位置移動
- マスコットを右クリック: メニュー表示
- マスコットをダブルクリック: ブルブルしてランダムなヘルプ表示

セーブデータ保存先:
- Windows: %APPDATA%\DesktopHatter\
- macOS: ~/Library/Application Support/DesktopHatter/

アンインストール:
- お使いのOSフォルダごと削除してください
- ユーザーデータも削除する場合は上記セーブデータフォルダも削除してください

注意:
- 署名なし配布の場合、Windows/macOSでセキュリティ警告が出ることがあります。
- 実行ファイルと .pck ファイルは同じフォルダに置いてください。
