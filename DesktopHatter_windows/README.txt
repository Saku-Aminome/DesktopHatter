DesktopHatter Distribution Folder (Windows)

This folder is ready for distribution.

Included files:
- DesktopHatter_windows.exe
- DesktopHatter_windows.pck
- README.txt (this file)

How to run:
1. Double-click `DesktopHatter_windows.exe`.
2. If Windows SmartScreen appears, click:
   `More info` -> `Run anyway`.

Basic controls:
- Drag: move mascot
- Right-click: open menu
- Double-click mascot: shake + random help tip
- Menu: Dress-up / Reminders / News / Language / Quit

Save data location:
- `%APPDATA%\DesktopHatter\`
- Example:
  `C:\Users\<username>\AppData\Roaming\DesktopHatter\`

Main saved files:
- hats.json
- reminders.json
- face_style.cfg
- window_state.cfg

Uninstall:
1. Delete this distribution folder.
2. To remove all user data, also delete:
   `%APPDATA%\DesktopHatter\`

DesktopHatter 配布フォルダ (Windows)

このフォルダは配布用です。

同梱ファイル:
- DesktopHatter_windows.exe
- DesktopHatter_windows.pck
- README.txt (このファイル)

起動方法:
1. `DesktopHatter_windows.exe` をダブルクリックして起動します。
2. Windows SmartScreen の警告が表示された場合:
   `詳細情報` -> `実行` を選択してください。

基本操作:
- ドラッグ: マスコットを移動
- 右クリック: メニューを開く
- ダブルクリック: ブルブルしてヒントを表示
- メニュー: きがえる / リマインダー / ニュース / 言語 / 終了

セーブデータ保存先:
- `%APPDATA%\DesktopHatter\`
- 例:
  `C:\Users\<ユーザー名>\AppData\Roaming\DesktopHatter\`

主な保存ファイル:
- hats.json
- reminders.json
- face_style.cfg
- window_state.cfg

アンインストール:
1. 配布フォルダを削除します。
2. ユーザーデータも削除する場合は次も削除します:
   `%APPDATA%\DesktopHatter\`
